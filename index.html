Rancang Asesmen Bermutu Tanpa Ribet

Platform asisten cerdas guru Indonesia untuk merancang blueprint, rubrik penilaian, lembar soal, dan kunci pembahasan Kurikulum Merdeka secara praktis, terstandarisasi, dan instan.

Mode Guest (Tamu)

Gunakan aplikasi secara instan tanpa pendaftaran akun. Seluruh data prompt atau produk asesmen diproses langsung dan API Key Anda disimpan aman di browser lokal Anda.

Mode Member

Daftarkan diri Anda untuk menyimpan seluruh riwayat hasil generate dokumen ke database awan Google Sheets. Admin akan menyetujui akses akun pendaftaran baru Anda.

Lengkapi parameter di bawah ini. Anda dapat menghasilkan format kisi-kisi (Blueprint), naskah soal lengkap, rubrik penilaian, kunci, serta pembahasan kognitif secara terstruktur.
